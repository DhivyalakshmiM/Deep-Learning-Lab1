# Deep Learning Laboratory

This repository contains Deep Learning laboratory experiments implemented using Python, TensorFlow, and Google Colab.

## Experiments

- Lab 1 – Perceptron Learning (AND/OR Gate)
- Lab 2 – Multilayer Perceptron (MLP)
- Lab 3 – Hyperparameter Optimization
- Lab 4 – Deep Neural Network (MNIST Digit Classification)

## Technologies Used

- Python
- TensorFlow / Keras
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Google Colab

## Dataset

The MNIST dataset is not included because of GitHub file size limitations.

Download it from:
https://www.kaggle.com/datasets/oddrationale/mnist-in-csv

- mnist_train.csv
