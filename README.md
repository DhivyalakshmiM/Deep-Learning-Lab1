# Deep Learning Laboratory

This repository contains Deep Learning laboratory experiments implemented using Python, TensorFlow, and Google Colab.

## 📚 Experiments

### Lab 1 – Perceptron Learning
- Implemented the Perceptron Learning Algorithm
- Trained on AND and OR gate datasets
- Visualized the decision boundary

### Lab 2 – Multilayer Perceptron (MLP)
- Solved the XOR problem
- Compared Sigmoid and ReLU activation functions
- Performed hyperparameter tuning

### Lab 3 – Advanced Hyperparameter Optimization
- Conducted manual Grid Search
- Compared learning rate, dropout, hidden units, and batch size
- Selected the best-performing model

### Lab 4 – Deep Neural Network (DNN)
- Classified handwritten digits using the MNIST dataset
- Generated a confusion matrix
- Performed error analysis
- Added an interactive digit prediction feature

## 🛠 Technologies Used

- Python
- TensorFlow / Keras
- NumPy

- Scikit-learn
